FAQ

Häufig gestellte Fragen

Ist Panum Grades wirklich kostenlos?
Ja, komplett. Es gibt keine versteckten Kosten, keine Premium-Funktionen, keine Werbung. Alles ist von Anfang an enthalten.
Sind meine Daten sicher? Wo werden sie gespeichert?
Deine Daten verlassen nie dein Gerät. Alles wird im lokalen Speicher deines Browsers (localStorage und Browser-Extension-Storage) gespeichert. Es gibt keinen Server, keine Datenbank, keine Cloud. Wir können physisch nicht auf deine Daten zugreifen.
Welche Browser werden unterstützt?
Google Chrome, Microsoft Edge und Mozilla Firefox. Da Edge auf Chromium basiert, funktioniert die Chrome-Web-Store-Version sowohl für Chrome als auch für Edge.
Funktioniert es auf dem Handy?
Aktuell ist Panum Grades eine Desktop-Browser-Erweiterung. Mobile Browser-Erweiterungen sind eingeschränkt, aber Firefox auf Android kann Add-ons installieren, daher könnte es dort teilweise funktionieren.
Kann meine Schule sehen, dass ich es benutze?
Nein. Panum Grades verändert nur, was du in deinem Browser siehst. Es sendet keine Daten an schulNetz oder anderswohin. Für die Server der Schule sieht dein Browser ganz normal aus.
Verändert es meine echten Noten?
Absolut nicht. Panum Grades liest nur die Noten, die schulNetz dir bereits anzeigt. Der Rechner und Simulator sind rein visuelle Werkzeuge — es wird nichts an schulNetz zurückgeschrieben.
Ist es mit schulNetz verbunden?
Nein. Panum Grades ist ein unabhängiges Projekt. Es ist nicht mit schulNetz, Centerboard AG oder irgendeiner Schule verbunden, endorsed oder affiliiert.
Wie deinstalliere ich es?
Rechtsklick auf das Extension-Icon in der Browser-Toolbar und «Erweiterung entfernen» wählen (Chrome/Edge) oder unter Add-ons entfernen (Firefox). Alle lokalen Daten werden beim Entfernen der Erweiterung gelöscht.
